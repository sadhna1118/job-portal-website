QUICK DEPLOY STEPS
===============================================

1. Open PowerShell in this folder

2. Run these commands ONE BY ONE:

git init
git add .
git commit -m "Initial commit: Job Portal by Sadhna"

3. Go to GitHub.com and create new repository:
   - Name: job-portal-web
   - Public or Private (your choice)
   - DON'T initialize with README
   - Click "Create repository"

4. Copy the commands GitHub shows you, OR run:

git remote add origin https://github.com/YOUR_USERNAME/job-portal-web.git
git branch -M main
git push -u origin main

(Replace YOUR_USERNAME with your actual GitHub username)

5. When asked for password, use Personal Access Token:
   GitHub.com ‚Üí Settings ‚Üí Developer settings
   ‚Üí Personal access tokens ‚Üí Generate new token
   ‚Üí Check "repo" ‚Üí Generate ‚Üí Copy the token
   ‚Üí Use as password

DEPLOY LIVE (Optional)
===============================================

After pushing to GitHub, deploy live on Render:

1. Go to render.com
2. Sign up with GitHub
3. New ‚Üí Web Service
4. Select your repo
5. Build: pip install -r requirements.txt && npm install && npm run build:css
6. Start: gunicorn app:app
7. Environment: Add SECRET_KEY variable
8. Create Web Service

Your app will be live in 5-10 minutes!
